# Netflix-Content-And-Customer-Analytics-PowerBI
Netflix Analytics Dashboard

Project Overview

This Power BI dashboard provides insights into Netflix content by analyzing movies and TV shows across genres, ratings, countries, and release years. The dashboard enables users to explore content trends and understand Netflix's content distribution.

Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling

Key Features

- Movies vs TV Shows Analysis
- Genre Distribution
- Content Ratings Analysis
- Release Year Trends
- Country-wise Content Distribution
- Interactive Filters and Navigation
